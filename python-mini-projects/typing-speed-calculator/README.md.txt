# Typing Speed Calculator

Welcome to the **Typing Speed Calculator**, a simple and fun way to test and improve your typing speed and accuracy!

## Features
- Multiple difficulty levels: Easy, Medium, Hard, and Expert.
- Calculates:
  - Typing speed in Words Per Minute (WPM)
  - Accuracy percentage
  - Total errors and time taken
- Expert mode hides your typing input for added challenge.

## How to Run
1. Make sure the following files are in the same directory:
   - `easy.txt`
   - `medium.txt`
   - `hard.txt`
2. Download and run the script `main.py`.
